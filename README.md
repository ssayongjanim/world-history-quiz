# 🌍 세계사 기말고사 대비 웹사이트

고2 1학기 기말고사 세계사 범위를 대비하기 위한 대화형 웹사이트입니다.

## 📚 학습 범위

- **1단원**: 명·청·임·이슬람·무굴·오스만
- **2단원**: 근대 유럽의 혁명과 변화
- **3단원**: 산업혁명과 제국주의
- **4단원**: 동양의 근대화
- **5단원**: 세계대전과 혁명
- **6단원**: 냉전과 탈냉전

## ✨ 주요 기능

### 1. 📝 문제 풀이
- 6개 단원 중 선택하여 문제 풀기
- 단원별 또는 전체 범위 선택 가능
- 문제가 매번 다른 순서로 출제됨 (셔플)
- 실시간 타이머 (문제당 2분)

### 2. ✅ 정답 확인 및 점수
- 풀이 완료 후 정답률 및 점수 확인
- 각 문제별 상세한 해설 제시
- 정오답 여부 명확하게 표시

### 3. 🔖 복습 기능
- 복습이 필요한 문제에 북마크 가능
- 북마크된 문제만 모아서 복습 가능
- 틀린 문제 중심 학습

### 4. 📊 성적 분석
- 전체 정답률 확인
- 푼 문제 수 및 정답 수 통계
- 단원별 정답률 비교
- 약한 단원 파악 및 집중 학습 가능

### 5. 💾 진행 상황 자동 저장
- 브라우저 로컬스토리지에 자동 저장
- 다시 방문해도 이전 진행 상황 유지
- 북마크 정보 보존

## 🚀 사용 방법

### 홈 화면
1. 6개 단원의 개요 확인
2. "모든 단원 풀기" 또는 "특정 단원 선택해서 풀기" 선택

### 문제 풀이
1. 풀 단원 선택 (복수 선택 가능)
2. 시작하기 버튼 클릭
3. 각 문제에서 선택지 클릭
4. 북마크 필요시 별(⭐) 버튼 클릭
5. 다음 버튼으로 다음 문제로 진행

### 복습 및 성적 분석
- 복습하기: 북마크된 문제 확인 및 재풀이
- 성적분석: 단원별 정답률 그래프로 확인

- 총 42개 문제
- 단원별 6~7개 문제

## 💡 팁

- 처음에는 1-2개 단원만 선택해서 시작하기
- 틀린 문제는 반드시 복습하기
- 단원별 정답률이 70% 이상이 되도록 목표 세우기
- 시험 전날은 북마크된 문제 집중 복습
